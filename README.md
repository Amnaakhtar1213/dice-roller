## DICE ROLLER

Simple game of dice roller built with the help of **HTML**, **CSS**, and **JavaScript**.In this project i learn how to work with array and innerHTML, that is just like a html which we code in JS.

----

## fEATURES

1. Button to roll dice 
2. After rolling we see a result in one section and also there is dice images that shown your number.
3. In this game we also roll more than one dice.
4. We roll 3,4,5 dice at once .
